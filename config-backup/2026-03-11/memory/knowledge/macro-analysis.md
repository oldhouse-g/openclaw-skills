## 一、核心分析镜片（从公设直接推导）

### 1. 中心化vs分布式判断镜片（公设一）

面对任何政策、技术变革或市场事件，第一个问题：

> **这件事是在加强中心化，还是在推动分布式？**

- **加强中心化**（产业政策集中资源、行业整合、平台垄断）→ 短期高效、GDP见效快，但增加系统脆弱性
- **推动分布式**（反垄断、扶持中小企业、去中心化金融）→ 短期效率下降，但增加系统韧性和创新活力

**螺旋规律：** 健康的经济体在两者之间螺旋摆动。长期只推中心化→积累脆弱性→大调整；长期只搞分布式→国际竞争力不足。

### 2. 需求压强驱动资本流动（公设二+三）

> **钱永远流向需求压强最大的地方。**

- 刚性需求（吃穿住行医）→ 需求压强稳定，对应防御性板块
- 弹性需求（社交、地位、娱乐、新奇）→ 需求压强随经济周期和技术变革大幅波动
- **关键判断：** 当前周期阶段，哪类需求压强在升高？哪类在衰减？

### 3. 技术=约束边界的改变者（公设三）

> **分析技术趋势时，不问"多炫酷"，问"它改变了什么约束？"**

- AI改变了**信息处理成本**的约束 → 金融、法律、教育、医疗诊断等行业受冲击
- 新能源改变了**能源供给**的约束 → 产业链重构
- 不改变任何实质性约束的技术 → 大概率是伪风口

## 二、宏观趋势分析四步法

### 第一步：约束分析（源自公设一）
- 当前的核心约束是什么？（资源、能源、劳动力、技术瓶颈、制度瓶颈）
- 约束在变松还是变紧？
- 谁在争夺这些有限资源？（国家间、企业间、阶层间）

### 第二步：需求分析（源自公设二）
- 人口结构变化 → 哪些需求在结构性增长/萎缩？
- 收入分配变化 → 消费升级还是消费降级？
- 安全感变化 → 储蓄率、风险偏好的方向？
- **底层刚性需求变化极慢，上层弹性需求变化极快**

### 第三步：经济机制分析（源自公设三）
- 价格信号有没有被扭曲？（价格管制、补贴、汇率干预）
- 资本流动是否顺畅？（金融管制、跨境壁垒）
- 债务周期处于什么位置？（扩张期→边际递减→出清期）
- 利润分配是否可持续？（分配不均积累到什么程度）

### 第四步：制度与文化约束（源自第四层）
- 政策方向是偏中心化还是分布式？
- 国际关系格局如何影响贸易和资本流动？
- 文化/社会心理（信心、预期）处于什么状态？

## 三、关键预警信号（从M理论推导）

| 信号 | M理论解释 | 预示什么 |
|------|-----------|---------|
| 中心化组织持续膨胀、效率指标反而下降 | 耗散结构：维持中心化的能量消耗大于产出 | 结构性危机临近 |
| 债务增速远超GDP增速 | 对未来的预支超过了实际创造能力 | 债务周期拐点 |
| 技术突变改变核心约束 | 约束边界被重新定义 | 产业格局大洗牌 |
| 分配严重失衡 | 公设二：公平感是进化沉淀的刚性需求 | 社会/政策调整压力 |
| 价格信号长期被扭曲 | 资源错配持续累积 | 被迫出清风险 |
