# 蟹大爷的投资估值理论

> 来源：TouZ理论笔记。蟹大爷用第一性原理推导PE/PB/ROE关系，形成自己的M估值框架。

## 核心估值公式推导

### 起点：投资者收益视角

设分红比例d、增长率g、折现率(最低预期回报)w、当前利润E₀

内在价值 V = d×E₀×(1+g) / (w-g)

PE = V/E₀ = d×(1+g) / (w-g)

### 关键认知

- PE的倒数1/PE≠投资收益率（只在零增长+全分红的特殊情况下才成立）
- 投资者实际收益率 = 股息率 + g
- PE是估值概念（我为利润支付多少倍价格），不直接等于回报
- 公式对分红比例、增长率、折现率的微小变化极其敏感→理解意义>计算意义→需要足够的安全边际

### PE的适用边界

- 零增长+全分红：1/PE ≈ 收益率基准
- ROE = WACC时（无超额价值）：合理PE ≈ 1/w
- PE > 1/w：有增长溢价
- PE < 1/w：有下跌折价
- g接近或超过w时：PE趋向无穷，失去意义→需要PEG

## PB与ROE的关系

### 公式推导
PB = ROE × PE
ROE = PB/PE

PB = d×ROE×(1+g) / (w-g)
低增长近似：PB ≈ (ROE-g) / (w-g)

### 关键认知

- ROE > w → PB > 1（公司在创造价值）
- ROE < w → PB < 1（公司在毁灭价值）
- 破净（PB<1）要区分：是真便宜还是ROE太低的折价
- ROE是市场估值的锚

### PE vs PB的使用场景
- PE：从投资者角度看收益→看收益率情况
- PB：从公司角度看资产→看公司实际情况
- 轻资产行业：PB偏大（软性资产被低估）→ROE显得大→用ROE看竞争力/护城河
- 重资产行业：用PB看资产价值和质量→资产是竞争力/护城河

## ROE/ROA/ROIC三视角

### ROE（股东视角）
- ROE = 净利润/股东权益
- "我投进去的钱，公司帮我赚了多少？"
- 杜邦拆解：ROE = 净利率 × 资产周转率 × 权益乘数
- 注意：财务杠杆会放大ROE，不一定反映业务好坏

### ROA（公司视角）
- ROA = 净利润/平均总资产
- "我掌管的所有家当，整体利用效率如何？"
- 不区分资本来源→掩盖利息成本和再融资风险
- 重资产行业ROA偏低、轻资产偏高→不能直接横向比

### ROIC（生意视角）
- ROIC = NOPAT/投入资本
- "这门生意本身的赚钱能力强不强？"
- 剔除了财务杠杆和无息负债的影响
- **ROIC > WACC = 价值创造的试金石**
- 长期ROIC > WACC → 通常拥有强大护城河

## 可持续增长率公式

g = ROE × (1-d)

其中d=分红率，(1-d)=利润留存率

### 前提条件
- 不增发新股（所有增长来自内部留存）
- 维持目标资本结构不变
- 维持经营效率不变

### 诊断工具
- 实际g >> 计算g → 财务杠杆提高或发新股稀释
- 实际g << 计算g → ROE不实或生意恶化

## 估值模型体系

### 三大绝对估值方法
1. **股利贴现DDM**：V = d×E₀×(1+g)/(w-g)
2. **现金流折现DCF**：将分红换成自由现金流
3. **剩余收益模型RIM**：基于超额收益

### DDM的股息率讨论

由公式变形：w = 股息率 + g
- 零增长时：w = 股息率（股息率直接=收益率）
- 有增长时：股息率+增长率=收益率
- 高股息率本身意味着比较确定的收益

### 股息持续性检查清单
- 股息支付率30~60%（<20%回报意愿不够，>80%持续性问题）
- 经营现金流净额需持续覆盖分红总额
- 多年ROE稳定且>预期收益率w
- 靠融资/变卖资产支付股息=危险信号

## M估值框架（蟹大爷总结）

### M估值的理解
- 1/PE = 基于当前盈利和估值的静态盈利收益率（不等于投资收益率）
- 股息率+g = 理论上完备的预期投资收益率
- PE₂ = d×(1+g)/(w-g) = 考虑实际收益后的重估PE
- PE₁ vs PE₂ 的比较 = 判断高估/低估
- 不应存在1/PE < 股息率的情况→出现时分析原因避免高息陷阱

### M估值五步骤
1. **设定主观标尺w**：个人明确的原则和投资纪律
2. **收集客观PE和股息率数据**：了解估值水平和确定回报
3. **分析预判g**：最难、需要综合判断的部分
4. **比较决策**：
   - 参考一：股息率+g > w
   - 参考二：PE₁ < PE₂（当前市场估值低于理论估值）
5. **风险排查**：确认是否存在股息率 > 1/PE的异常

## 与M理论的关联

### ROE/g/w与M理论的对应
来自M理论微观应用层：
- ROE = 企业占据"经济水网"有利位置的量化体现
- g = 企业持续拓宽"经济河道"的能力
- w = 市场对未来现金流确定性的风险评估

### 投资方法论的底层逻辑
- 从第一性原理推导全部估值关系（数豆子方法论）
- 区分三个视角（ROE/ROA/ROIC）= 相似系统思维（同一事物多角度）
- 强调"公式的理解意义>计算意义"= 真正的理解来自直觉而非形式
- g = ROE×(1-d) 作为诊断工具 = 最小完备系统（一个公式包含核心信息）
