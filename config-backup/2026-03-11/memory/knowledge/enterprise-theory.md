## 从公设出发的推演链

### H1. 企业存在的理由 ✅ 必然

```
分工深化+交换频繁（E2）
+ 每次交换都有成本（寻找对手、谈判、监督、执行）= 交易成本
+ 某些交易反复发生且交易成本较高
→ 把反复的交易"内部化"→ 成本更低
→ 企业 = 将一组反复发生的市场交易内部化的组织
→ 企业边界 = 内部化成本 < 市场交易成本的范围
→ 超出范围：内部协调成本 > 市场交易成本 → 应该外包
```

科斯定理（1937），逻辑严密。从公设一（有限性约束→交易成本不可避免）推出。

### H2. 企业=中心化组织在经济中的具体形态 ✅ 必然

```
企业将交易内部化（H1）
+ 内部化需要决策权集中+指令链+层级管理
→ 企业天然是中心化组织
→ 中心化的优劣势在企业中全部体现：
   优势：高效、标准化、规模经济
   劣势：灵活性不足、创新受限、规模越大越脆弱
```

### H3. 企业竞争力的三个维度 ✅ 高概率

```
企业在市场网络中竞争（E8）
→ 竞争力 = 三个维度的综合：
   a) 网络位置：处于价值链的稀缺性环节（E6）→ 定价权
   b) 技术优势：第三种竞争优势（F1）→ 做同一件事比对手更好
   c) 组织优势：中心化效率 or 分布式灵活 → 形式匹配业务需求

→ 三者中，网络位置（稀缺性）最为关键——
   技术和组织优势都可能被追赶，但稀缺性+壁垒更持久
```

### H4. 企业规模的约束 ✅ 必然

```
企业规模扩大→内部协调成本上升（层级增多、信息损耗、官僚化）
+ 中心化组织的脆弱性随规模增加（公设一）
→ 存在一个最优规模：内部化收益=内部化成本的平衡点
→ 超过这个规模→效率下降→应该拆分或外包
→ 技术可以改变这个平衡点：
   信息技术降低内部协调成本→企业可以更大（平台经济的巨型企业）
   模块化降低外部交易成本→企业可以更小（专业化小公司+外包网络）
```
