# 蟹大爷的电磁场问题解法体系

> 来源：《如何理解传输线》《电磁场》《天线》

## 核心方法论（一句话）

> "绝大部分电磁场问题的解，都是把方程转化成'振子方程'的形式，然后求解！"

## 整体思维导图

电磁场问题分四大块，每块最终都归结为振子方程：

1. **传输线**（微波/射频电路基础）→ 电报方程 → 正弦信号假设 → 振子方程
2. **无源静场**（场分布问题基础）→ 拉普拉斯方程 → 分离变量 → 三个振子方程
3. **无源电磁场**（波导/本征问题基础）→ 赫姆赫兹方程 → 平面波简化 → 振子方程
4. **有源电磁场**（辐射/天线基础）→ 非齐次赫姆赫兹 → 格林函数 → 球坐标 → 振子方程

## 传输线

### 核心认知
> "传输线只不过是考虑了寄生参数的普通模拟电路！"

寄生参数：串联（电感L、电阻R）+ 并联（电容C、导纳G）

### 电报方程
由基尔霍夫定律直接得到：
- ∂V/∂z = -(R+jωL)I（电压差=串联阻抗压降）
- ∂I/∂z = -(G+jωC)V（电流差=并联导纳分流）

### 两个关键参数

**传播常数γ**（=α+jβ）：
- 表征系统传输特性
- 对应振子方程中的常数项
- 包含系统全部传输参数
- 不同系统只需替换γ对应的参数即可得到新系统特性

**特性阻抗Z₀**：
- Z₀ = √((R+jωL)/(G+jωC))
- 包含全部寄生参数
- 侧重电路/匹配问题
- 无耗时Z₀ = √(L/C)，与频率无关（非色散）

### 左手传输线
只需交换L和C的位置：
- 串联寄生阻抗变为1/jωC
- 并联寄生导纳变为1/jωL
- 代入传播常数公式 → 群速正、相速负 = 左手特性
- **不需要推导新方程，只替换参数！**

### 巴伦（Balance-Unbalance）
- 完成单端传输到差分传输的变换
- 高频区通过传输线介质完成（电磁波传输方式）
- 低频区按普通变压器工作（磁场耦合）
- 差模信号：传输线模式（磁芯不起作用）
- 共模信号：变压器模式（磁芯耦合）

## 电磁场

### 场的概念
- 场是为了解决"超距作用"引入的假设
- 解决了瞬时动量/能量不守恒的问题
- 库仑力→除掉测试电荷→电场E
- 是否真实存在"以太"？引入场后不需要考虑了，但不确定是否存在

### 散度和旋度
**只有两种基本场类型**：
1. 散度场：从源直线向外（如点电荷电场）→ 通量→散度
2. 旋度场：围绕源旋转（如稳恒电流磁场）→ 环量→旋度

现实中所有场 = 这两种的组合。散度+旋度完备描述所有场。

**散度**：通量/体积的极限（标量）→ 源的强度
**旋度**：环量/面积的极限（矢量，取最大值方向）→ 旋转源的强度

### 麦克斯韦方程组
四个方程本质是描述电场和磁场的源：
1. ∇×H = J + ∂D/∂t（磁场的旋度源 = 电流+位移电流）
2. ∇×E = -∂B/∂t（电场的旋度源 = 变化的磁场）
3. ∇·D = ρ（电场的散度源 = 自由电荷）
4. ∇·B = 0（磁场没有散度源 = 无磁单极）

### 蟹大爷的独特理解

**"电是更根本的存在"**：
- 电场有两种源（散度+旋度），磁场只有旋度源
- 磁场的全部源都来自电的转换
- 蟹大爷个人观点：磁场是电场传递过程的"褶皱"，对应电场的时延

**位移电流是麦克斯韦最闪光的地方**：
- 没有位移电流→电磁波无法传播（电生磁→磁生电的链条"断"掉）
- 有了位移电流假说→相对论的出现是早晚的事

**从位移电流到相对论的推理**（蟹大爷原创理解）：
- 力学方程：m d²x/dt² + r dx/dt + kx = F
- 电学方程：L d²q/dt² + R dq/dt + q/C = V
- 位移电流出现后，电学第一项(q的变化)可以在无电荷移动时发生
- 力学中x不能跳变→只能是质量m变成变量→这就是相对论结果

### D、E、H、B的区分
- E：场能力（和介质有关，实际测量的场）
- D：源能力（和介质无关，源本身的特性）
- B：场能力（和介质有关，实际测量的磁场）
- H：源能力（和介质无关，因历史原因被叫"磁场强度"）

实际应用中更常用的是场能力量（E和B），但仿真中磁场常用H导致混淆。

**SAR案例**：天线末端电场E大但SAR不是热点→因为液体介电常数高使E在内部变小→而B不受介质影响→磁场感应的电场才是SAR热点

### 从源到电磁波
时变电荷→产生电场→变化的电场成为磁场源→磁场又产生电场→无穷嵌套=电磁波

散度场（与r²成反比）比旋度场（与r成反比）衰减更快→远场只剩旋度对应的辐射场。

### 矢量分析
蟹大爷的简化理解：
- 矢量的加法和乘法搞清楚→全部矢量分析就有了
- **矢量代数积 = -点积 + 叉积**（核心公式，全部矢量运算的反复应用）
- 哈密顿算符∇当做普通三维矢量处理
  - ∇作用于标量 = 梯度
  - ∇·矢量 = 散度
  - ∇×矢量 = 旋度
- 所有电磁场矢量分析 = 哈密顿算符 + 代数积公式的反复套用

### 电磁场求解总体逻辑
1. 首先看源是什么状态→决定麦克斯韦方程组变形成什么样
2. 然后把方程转化为振子方程形式→求解

### 无源电磁场→赫姆赫兹方程
消去一个变量→三维波动方程→时谐场→赫姆赫兹方程（形式上是振子方程）

### 平面电磁波
假设场只有一个分量→三维→一维→振子方程→直接求解
**任何电磁波都可以表示成平面波的叠加**→理论上任何无源电磁场可解

### 格林函数（点源的场）
- 用球坐标把三维→一维（只和r有关）
- 源外区域齐次→振子方程→标准解
- 格林函数 = 空间冲击响应的场分布
- 任意源的场 = 格林函数和源函数的积分（叠加原理）

### 蟹大爷的感悟
> "梳理清楚电磁场问题后我稍有点失望：人们这么多年开发的数学工具，还是有挺大局限，这么个电磁场问题，还不得不反复来套用一个已有解的方程。"

这句话本身就是"数豆子"方法论的最好注脚——物理的底层确实就是那几个"豆子"的反复组合。

## 天线

### 核心认知
> "天线本质上是电子的'秋千'"

### 秋千的特性→天线的特性
秋千（振子）的核心特点：
- 只需很小的作用力就能让它晃起来
- 运动损耗很小
- 运动具有周期性
- 很小的能量、运动幅度大
- 天然具有"弱作用力/弱信号放大"的特性

### 天线的基本原理
天线 = 振荡的电偶极矩
外在能量支持电荷振荡→源源不断产生相互嵌套的电场和磁场往外扩散 = 电磁波

### 偶极子场计算
从有源电磁场的格林函数方法出发：
1. 短偶极子（l << λ）上的电流产生磁矢位A
2. 由A计算磁场H
3. 由H计算电场E
4. 得到远场辐射场和近场存储场

### 蟹大爷的专栏构想："振子看世界"

专栏内容规划：
1. 天线是电子的秋千
2. 电磁场是空间上的振子
3. 传输线和电磁波是振子的级联
4. 材料参数是原子振子的特性
5. 量子力学中的量子是振子的推广

## 数理方程基础

### 三大代表性偏微分方程
1. 波动方程（双曲型）
2. 热传导方程（抛物型）
3. 拉普拉斯/泊松方程（椭圆型）

### 不同坐标系下的特殊函数
- 球坐标→勒让德函数（球谐函数）
- 柱坐标→贝塞尔函数
- 都是拉普拉斯方程分离变量后得到的

### 信号处理基础
- 傅里叶级数：周期信号→离散频谱
- 傅里叶变换：非周期信号→连续频谱
- 拉普拉斯变换：傅里叶变换的推广（加衰减因子处理不收敛信号）
- DFT/FFT：离散化的傅里叶变换
- Z变换：离散化的拉普拉斯变换
- 卷积：系统冲击响应和信号的积分（物理意义：t时刻的响应=之前所有响应在t时刻剩余量的叠加）

### 蟹大爷对学习方法的思考
> "先从树木开始好、还是先给个森林的全貌？"
> "大部分教科书从局部细节开始，学完时前面的已经遗忘，很难形成整体概念。"
> "我的目标是先用粗略方法把整体梳理一遍，形成整体印象，再进入细节。"

这是"追根溯源"方法论在教学中的应用——先看全貌（森林），再看细节（树木）。
