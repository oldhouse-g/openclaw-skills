# 振子方程详解

## 基本方程体系

### 标准振子方程（无阻尼自由振动）
m d²x/dt² = -kx
通解：x = Ae^{iω₀t}，其中 ω₀ = √(k/m)

### 阻尼自由振动
m d²x/dt² + r(dx/dt) + kx = 0
- 过阻尼（r² > 4km）：两个实数根，无振动
- 临界阻尼（r² = 4km）：两个等根，无振动
- 欠阻尼（r² < 4km）：复数根，逐渐衰减的振动（大部分关注的情况）

### 受迫振动（无阻尼）
m d²x/dt² + kx = F₀e^{iωt}
稳态响应：x = Ce^{iωt}
C = F₀/m(ω₀²-ω²)

关键点：
- ω < ω₀：C为正，响应与激励同相
- ω > ω₀：C为负，响应与激励反相（180°）→ 左手材料的微观解释
- ω = ω₀：C无穷大 = 谐振（音障的本质）

### 阻尼受迫振动（全振子方程）
m d²x/dt² + r(dx/dt) + kx = F₀e^{iωt}

响应幅度：|C| = F₀ / m√((ω₀²-ω²)² + r²ω²)

响应落后激励的相差：
Δφ = arctan(rω/(ω₀²-ω²))

**这个相差表达式是理解大量物理问题的关键**——
材料特性、折射率、损耗、左手材料、波速变化，全部由它决定。

## 损耗的物理本质

### 推导过程（蟹大爷原创理解）

理想振子假设弹簧"固定端"完全不动：x = Δx（位移=压缩量）

现实中固定端会移动 → 质点位移x和弹簧压缩量Δx之间有相差Δφ：
- Δx = x × cos(Δφ) + (dx/dt) × sin(Δφ)/ω₀

推导后得到标准阻尼方程：
m d²x/dt² + (k sin(Δφ)/ω₀)(dx/dt) + k cos(Δφ)x = 0

其中阻尼系数 = k sin(Δφ)/ω₀

**关键洞见**：
- "摩擦力和速度成正比"看来是表象，不是本质原因
- 损耗的真正来源：固定端（原子核）吸收了部分能量→热运动加剧
- 这统一了损耗、热和噪声

## 振子的能量

外力做的功：P = F × dx/dt

稳态下：外力的功率只和损耗项有关（储能项动能+势能的和恒定）

Q值 = 2π × 总储能 / 每周期做功
在谐振点：Q = ω₀m/r

另一个视角：功率取决于响应和激励之间的相差
- 相差90°时→做功最多
- 只要非正交就有做功
- 达到正交→维持平衡

## 从振子到传输线

### 级联LC电路

LC谐振电路 = 电学最小完备系统
级联LC → 用电容做中间级联元件（因为q直接产生V，不需要额外变换）

前后级相差可用R+C等效 → 串并转换 → 得到基本传输线等效电路 → 电报方程

### 力学与电学的参数对应

| 力学 | 电学 | 物理意义 |
|------|------|---------|
| 质量m | 电感L | 惯性 |
| 位移x | 电荷q | 基本物理量 |
| 阻尼系数 | 电阻R | 损耗 |
| 弹簧常数k | 1/C（电容倒数）| 恢复力 |
| 外力F(t) | 电压V(t) | 激励 |

蟹大爷指出：电学中的R、L、C是把作用特性等效为元件的结果。力学中对应量是分布参数（弹簧同时有质量、弹性和损耗），不方便定义为独立元件。

## 从振子到材料

### 原子的振子模型
- 原子核 = 固定端
- 外层电子 = 质点
- 原子间电场 = 弹簧

### 材料中波矢的推导

第一个原子电子运动→相差Δφ（振子响应）
电场力传递到第二个原子→时间延迟 L/c → 相差 ω₀L/c
第二个原子电子运动→又一个相差Δφ

总相差 = Δφ + ω₀L/c + Δφ
材料中波矢 = 真空波矢 + 总相差/L

### 材料特性全部来源于相差

Δφ = arctan(rω/(ω₀²-ω²))

- r（损耗）越大→Δφ越大→波速越慢
- ω₀越大（束缚越强）→Δφ越小→波速越快
- ω > ω₀ → Δφ < 0 → 左手材料（异常折射）

## 从振子到量子力学

### ω和k的统一本质

ω（角频率）：相位随时间变化的速率
k（波矢）：相位随空间变化的速率
两者在数学上完全等价，只是度量维度不同

关系：ω = v_phase × k（相速关系）
群速：v_group = dω/dk

### 从振子到薛定谔方程的推导路径

1. 振子方程解：f = Ae^{iωt}
2. 加入空间传播：ψ = Ae^{i(kx-ωt)}
3. 对t求导→得到ω的算符表达
4. 对x求导→得到k的算符表达
5. 代入E=ħω, p=ħk → 得到能量和动量算符
6. 代入经典能量守恒E=p²/2m+V → 薛定谔方程
7. 一维→三维：x→r̄, k→k̄ → 通用薛定谔方程

### 量子与经典的对应

| 经典振子 | 量子系统 |
|---------|---------|
| ω₀（谐振频率）| E/ħ（能级）|
| k（波矢）| p/ħ（动量）|
| 无限长弹簧→无固有频率 | 自由粒子→能量连续 |
| 有限长弹簧→固有频率 | 势阱→能量量子化 |
| 储能=动能+势能 | 哈密顿量H=T+V |

## 通用方程对照表

| 方程 | 物理含义 |
|------|---------|
| d²f/dx² + k²f = 0 | 振子/本征模 |
| ∂f/∂t = D∇²f | 热传导/扩散 |
| ∂²f/∂t² = c²∇²f | 波动 |
| ∇²f = 0 | 势（拉普拉斯方程）|
| ∇²f = -ρ/ε | 有源势（泊松方程）|

它们都可以从与振子方程的对比中得到理解。
