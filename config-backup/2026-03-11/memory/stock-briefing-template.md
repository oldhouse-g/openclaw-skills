# 股市晨报模板

## 数据源
- Tushare Pro（token已配置在~/.zshrc）
- A股大盘：pro.index_daily()（上证000001.SH、深证399001.SZ、创业板399006.SZ）
- A股个股：pro.daily()
- 港股：pro.hk_daily() 或腾讯行情接口备用

## 模板结构

### 1. 📊 大盘收盘概况
- 三大指数：收盘价、涨跌幅
- 沪市成交额、深市成交额、两市合计
- 近5日成交量趋势图（文本柱状图，标注放量/缩量）

### 2. 📋 自选股行情一览
- 🇨🇳 A股：各票收盘价、涨跌幅，标注跌幅最大/涨幅最大
- 🇭🇰 港股：同上
- （自选股列表待蟹大爷提供确认）

### 3. 📰 今日要闻提示
- 3-4条当日重要财经新闻
- 简明扼要，带编号

### 4. 💡 小结
- 整体市场分析
- **量能分析**：量价关系判断（放量杀跌/缩量企稳/放量突破等）
- **趋势判断**：基于近期量能给出短期方向性判断
- 今日关注点

## 格式规范
- 用 emoji 标识涨跌（📈📉）
- 涨跌幅带正负号
- 成交额用"万亿"为单位
- 柱状图用 █ 字符可视化
- 简洁有用，不废话
